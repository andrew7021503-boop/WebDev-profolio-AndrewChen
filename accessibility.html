Accessibility Statement

Accessibility Features on This Website

1. Semantic HTML

I used standard HTML structure and tags to make the page clear for screen readers, helping users with visual difficulties.

2. Responsive Layout

The website adapts to desktop, tablet and mobile phones. Users can browse easily without horizontal scrolling.

3. High Color Contrast

I chose clear color matching between text and background. Dark text on light background improves readability for people with low vision.

4. Keyboard Accessible

All navigation links and buttons can be used with keyboard Tab key, convenient for users who cannot use a mouse.

5. Readable Text

Proper font size, line height and paragraph spacing make text easy to read for all visitors.

6. Image Alt Text

Every image has descriptive alt text, so screen readers can explain what the images show.
