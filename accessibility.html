Accessibility Statement

Accessibility Features on This Website

1. Semantic HTML

I used standard HTML structure and tags to make the page clear for screen readers.

2. Responsive Layout

The website adapts to desktop, tablet and mobile phones.

3. High Color Contrast

Clear color matching improves readability for people with low vision.

4. Readable Text

Proper font size, line height and spacing make text easy to read.

5. Image Alt Text

Every image has descriptive alt text for screen readers.

6. Contact me

I have made a contact link for visiters to ask questions.
