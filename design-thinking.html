My Design Thinking Process

I used the five stages of Design Thinking to build this portfolio website step by step.

1. Empathize

I thought about website visitors. Users need clear navigation, readable text and comfortable layout on both computer and mobile phones.

2. Define

I made a clear goal: create a personal portfolio to show my identity, projects, ideas on global goals and accessibility features.

3. Ideate

I planned page structure: use a navigation bar to switch pages, apply card style for projects, and use soft colors for a friendly visual effect.

4. Prototype

I first built HTML structure, then added CSS styles. I divided content into different sections and adjusted layout for different screens.

5. Test & Iterate

I tested the website on desktop and mobile. I modified font size, spacing and colors to fix layout problems and improve user experience.
